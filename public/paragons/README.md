# Paragon Images

Place Paragon card images in this directory.

## File Naming Convention

Each image should be named with "Paragon " prefix followed by the Paragon name with a `.png` extension:

- `Paragon Abraham.png`
- `Paragon Judah.png`
- `Paragon Eve.png`
- `Paragon Rachel.png`
- `Paragon Reuben.png`
- `Paragon Joseph.png`
- ... (see full list below)
