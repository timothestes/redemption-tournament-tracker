# Paragon Images

Place Paragon card images in this directory.

## File Naming Convention

Each image should be named after the Paragon in lowercase with a `.png` extension:

- `abraham.png`
- `judah.png`
- `eve.png`
- `rachel.png`
- `reuben.png`
- `joseph.png`
- ... (see full list below)
